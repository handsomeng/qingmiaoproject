# 青苗计划 - VibeCoding 产品树结构

> 三层树状结构，帮助零基础用户理解「我能做出什么产品」

---

## 第一层：动机分类

1. 💰 **赚钱** - 我想做一个能卖钱的小工具
2. 🚀 **职场提效** - 我想做一个提升自己效率的工具
3. 🎮 **趣味性** - 我想做一个好玩/有创意的东西

---

## 第二层 + 第三层：完整穷举

### 💰 赚钱

#### 卖工具类
| 产品名称 | 一句话描述 |
|---------|-----------|
| 小红书封面生成器 | 输入标题，一键生成高点击率的封面图 |

| 表情包生成器 | 输入文字，自动配图生成表情包 |
| 情侣头像生成器 | 上传照片，生成 CP 感头像 |

| 星座运势生成器 | 每日星座运势自动生成，可做账号内容 |
